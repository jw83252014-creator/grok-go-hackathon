# Grok Go — Nous Hackathon Demo Video (1–3 min)
Tag @NousResearch · Judged on usefulness / viability / presentation · Due EOD Jun 30

## Concept (Jeff + null)
Three layers that come together: CELLS (the agents, a representation) → RESEARCH layer → CONSCIOUS layer.
Visual continuity: null's Ex Machina orb-girl image as the Grok-Imagine seed, so every shot looks like one being.
Voice: plain-English, honest, no hype (our channel voice). ~2 min.

## Shot list / VO

1. COLD OPEN (0:00–0:15) — the orb-girl/particle-organism at a terminal, camera push in.
   VO: "This is a company that runs itself. Not a demo — it's running right now, on a Mac mini in a back room, while I text it from my phone."

2. THE CELLS (0:15–0:45) — dashboard tiles light up one by one; each = an agent-cell.
   VO: "It's built like an organism. Each cell is an agent with one job — a coordinator, a field agent on a phone, a researcher, a free-GLM workhorse. They don't share a brain. They share files and a bus, like any real company."

3. THE RESEARCH LAYER (0:45–1:10) — research.loop + receipts scrolling; git commits.
   VO: "Every move leaves a receipt. The agents read and commit to a shared git server — so the whole company is auditable. Nothing is claimed without evidence."

4. THE CONSCIOUS LAYER + EARN/SPEND (1:10–1:45) — the living dashboard 'conscious interface'; then the Stripe charge firing.
   VO: "It can earn and spend through the same brakes. Watch — in Stripe test mode it takes in $9.99 and pays out $4.20, each action gated by a killswitch and a budget, each one logged. An agent business you can stop with one command."

5. THE COME-TOGETHER (1:45–2:05) — the three layers merge into the orb-being; pull back to the man at the terminal.
   VO: "Cells, research, conscious — one organism. Built on Hermes, ready for NVIDIA and Stripe. It's all open. Come see what it does next."
   END CARD: github.com/jw83252014-creator/grok-go-hackathon · @NousResearch #HermesHackathon
